# Quick Start

## Local Setup

```bash
git clone https://github.com/YOUR_USERNAME/orchestrator-core.git
cd orchestrator-core
docker-compose up
```

Then open http://localhost:3000 for dashboard.

## API Endpoints

- POST /task - Submit task
- GET /tasks - List tasks
- GET /task/:id - Get task status
- GET /agents - List agents
- GET /health - Health check
